# Hackatron: Cambio climatico, Discord bot                   
- Comandos (sketch):
9Ayuda (comando lista informacion basica sobre el bot),
9Origen (describe el mero origen del cambio climatico),
9Gravedad (comando explica porque es grave el cambio climatico),
9Mapa (mapa sobre las emisiones actuales),
9Estadisticas (algunas estadisticas simples  en texto sobre las emisiones actuales),
9Goals (introduce el protocolo de 17 goals, algo que piensa resolver el cambio climatico, asi como otras cosas, asi como el sitio web tiene informacion del tema),

Los comandos estan basados en el orden en el que el usuario usualmente buscara el tema, el bot es informativo mas que dar una solucion en si mismo, pero ayuda a acceder informacion.
